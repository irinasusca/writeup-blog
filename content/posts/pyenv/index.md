## pyenv

[Source](https://github.com/pyenv/pyenv-virtualenv).

 - To view venvs: `pyenv virtualenvs`
 
 - To activate venv: `pyenv activate venv-name`
 
 - To create venv with a certain downloaded version: `pyenv virtualenv v.x venv-name`
 
 - To install a py version: `pyenv install v.x`.
